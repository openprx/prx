# prx-pdk

The Rust crate that powers the [PRX Rust PDK](../README.md).

This crate provides:

- WIT bindings generated by `wit-bindgen` for all `prx:host/*` interfaces
- Ergonomic wrapper modules: `log`, `config`, `kv`, `http`, `memory`, `events`, `clock`
- Shared types: `ToolSpec`, `PluginResult`, `MiddlewareAction`, `HttpResponse`, `MemoryEntry`
- Macros and helpers used by plugin authors

## Usage

Plugin authors should depend on this crate (not reference it directly from the repo root):

```toml
[dependencies]
prx-pdk = { git = "https://github.com/openprx/openprx", subdirectory = "pdk/rust/prx-pdk" }
wit-bindgen = { version = "0.51", default-features = false, features = ["macros"] }
```
