# Web Console

Browser-based management interface for OpenPRX.

> **Status**: Planned — located at `web/` in the repository.

## Planned Features

- Real-time conversation monitoring
- Configuration editor
- Memory browser and search
- Cron job management
- Evolution dashboard and analytics
- Remote node status
- Provider/channel health overview
- Session and subagent management

## Tech Stack

- Svelte SPA
- Served by the OpenPRX gateway (embedded static files)
